# Superstore Sales Analysis – Excel Project

This project analyzes sales performance across regions, provinces, and product sub-categories using the **Superstore Sales Dataset**.  
It showcases essential **Business Analyst** skills such as data cleaning, formatting, sorting, filtering, pivot analysis, and converting raw data into meaningful business insights.

---

## Project Objective

As a sales manager for a large Canadian retail chain, the goal is to understand:

- Which product sub-categories generate the **highest profits**
- Which categories are **loss-making**
- Profit patterns across **regions and provinces**
- Insights that will help improve business strategy or discontinue low-performing products

---

## Dataset

**File:** `superstore_sales project.xlsx`

The dataset includes fields such as:

- Region & Province  
- Order & Ship Dates  
- Product Category & Sub-Category  
- Sales, Profit, Quantity  
- Customer Segment  

---

## Skills & Techniques Used (Excel)

- Data Cleaning  
- Formatting & Readability Enhancement  
- Filtering & Segment-wise Data Separation  
- Multi-Level Sorting  
- Conditional Formatting  
- Pivot Table Creation  
- Region-wise & Category-wise Profit Analysis  
- Identifying profit drivers & loss-making categories  

These skills are foundational for **Business Analyst** roles.

---

## Activity 1: Data Preparation

### Steps Completed

- Adjusted column widths  
- Styled the header row and added borders  
- Separated data for each customer segment  
- Focused analysis only on **Corporate** customers  
- Formatted Sales & Profit (USD + rounded)  
- Formatted Order & Ship Dates  
- Sorted data by Region → Province → Sales  
- Applied conditional formatting:  
  - Top 10% sales highlighted per province  
  - Profit heatmap (Green = high profit, Red = losses)  
- Added double borders to separate regions  

---

## Activity 2: Pivot Table Insights

### **1. Top 3 Most Profitable Sub-Categories (by Region)**  
A Pivot Table was used to compute total profit by sub-category within each region.  
From this, the **top 3 profit-generating** sub-categories were identified.

### **2. Most Loss-Making Sub-Categories**  
- Identified sub-categories with **negative profit**  
- Analyzed **worst-performing regions**  
- Noted business patterns (e.g., tables & chairs often bought together)
